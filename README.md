# 基智回测 (FundWise Backtest)

中国公募基金智能投资回测与组合推荐工具。

## 项目结构

```
fundwise-backtest/
├── backend/          # Java Spring Boot 后端
├── frontend/         # 纯HTML/CSS/JS 前端
├── scripts/          # Python 数据采集与预处理脚本
├── sql/              # 数据库建表脚本
├── docs/             # 项目文档 (设计、API等)
└── README.md
```

## 技术栈
- **后端**: Java 17 + Spring Boot 3.x
- **前端**: Vanilla HTML/CSS/JS + Chart.js
- **数据库**: MySQL 8.x / PostgreSQL 14.x
- **数据获取**: Python 3.9+ + akshare
- **构建工具**: Maven
