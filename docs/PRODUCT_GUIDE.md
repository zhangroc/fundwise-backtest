# FundWise 基智回测 - 产品功能介绍

## 一、产品概述

**基智回测（FundWise）** 是一款面向个人投资者的基金投资智能分析平台。

### 核心价值

- 📊 **数据驱动**：26,000+ 只基金，200万+ 条净值数据
- 🤖 **智能推荐**：根据风险偏好智能配置投资组合
- 📈 **历史回测**：基于真实历史数据验证投资策略
- ⚡ **高性能**：毫秒级响应，流畅用户体验

### 技术指标

| 指标 | 数值 |
|------|------|
| 基金总数 | 26,129 只 |
| 净值记录 | 2,027,179 条 |
| 数据跨度 | 25 年 |
| API响应 | < 50ms |
| 页面加载 | < 2s |

## 六、使用指南

### 快速开始

1. **访问首页**：打开浏览器访问 `http://192.168.0.2`
2. **筛选基金**：点击「基金筛选」，设置筛选条件，点击「开始筛选」
3. **查看详情**：点击基金行的 👁️ 图标，查看详细信息
4. **创建组合**：点击「添加到组合」，创建或选择现有组合
5. **执行回测**：在组合页面选择组合，设置参数，点击「开始回测」

### 最佳实践

1. **筛选策略**
   - 先按类型缩小范围
   - 再按业绩指标筛选
   - 最后查看详情深入分析

2. **组合构建**
   - 不同类型基金分散配置
   - 权重根据风险承受能力调整
   - 定期进行再平衡

3. **回测验证**
   - 选择足够长的时间段
   - 关注最大回撤是否可接受
   - 多次回测验证稳定性

## 七、常见问题

### Q1: 筛选结果加载慢怎么办？
A: 系统已优化，正常响应时间 < 100ms。如仍慢，请检查网络连接。

### Q2: 净值数据多久更新一次？
A: 数据来源于公开渠道，当前为历史快照数据，后续会支持实时更新。

### Q3: 回测结果能代表未来收益吗？
A: 历史业绩不代表未来表现，回测仅供投资参考，不构成投资建议。

### Q4: 支持哪些基金类型？
A: 目前支持股票型、混合型、债券型、指数型、QDII、货币型等主流类型。
